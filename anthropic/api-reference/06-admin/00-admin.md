# Admin

The Admin API provides programmatic access to manage your Anthropic organization. Use these endpoints to manage organizations, invites, users, workspaces, workspace members, API keys, usage reports, and cost reports.

All Admin API endpoints require an admin API key with appropriate permissions. Admin API keys can be created in the Anthropic Console.

## Available Resources

- **Organizations**: Retrieve information about your organization
- **Invites**: Create and manage invitations for new users
- **Users**: List, retrieve, update, and remove users in your organization
- **Workspaces**: Create and manage workspaces for organizing resources
- **Workspace Members**: Add, update, and remove members from workspaces
- **API Keys**: List, retrieve, and update API keys
- **Usage Report**: Retrieve usage metrics for messages and Claude Code
- **Cost Report**: Retrieve cost breakdowns for your organization
